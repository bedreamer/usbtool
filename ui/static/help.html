添加对象
使用鼠标左键点击左边工具栏中的对象，拖放到合适位置松开左键即可。
如何选中对象
  1. 鼠标单击需要选择的对象
  2. 使用鼠标左键按住对象不放，此时该对象被选中。
  3. 松开鼠标左键，选择的对象将会取消。
移动对象
  1. 选中对象
  2. 拖动选中的对象到需要的位置，松开鼠标左键即可。
快速添加对象
若需要连续快速添加某一种对象，只需要按照`添加对象`操作后按照如下流程操作即可：
  1. 使用鼠标左键按住想要快速添加的对象
  2. 使用键盘Ctrl+C复制该对象，后松开左键
  3. 将光标移动至需要放置对象的位置，使用键盘Ctrl+V即可快速重复添加
快速编辑对象
使用鼠标左键双击要编辑的对象，在弹出的窗口内编辑即可。
如何快速调整对象大小
  1. 选择对象
  2. 按住Ctrl不放
  3. 向左，向上拖动鼠标缩小对象；向右，向下拖动鼠标放大对象。
  4. 大小调整完成后松开鼠标即可。
如何旋转对象
  1. 选择对象
  2. 按住Ctrl不放，单次点击按键R逆时针旋转90度
  3. 按住Ctrl不放，单次点击按键T逆时针旋转1度
  4. 大小调整完成后松开鼠标,放开Ctrl即可。
删除对象
  1. 选择对象
  2. 使用键盘Delete删除
